# Coding otto robot with Blockly

## Usage
1. Open blockly
2. Connect USB cable to Otto
3. Choose microchip (Arduino Nano) and Select Port (COMX or USBX)
3. Load example
4. Play and explore
5. Disconnect USB cable 
6. Close Blocky
